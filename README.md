# 🏥 NutriCheck - 자가 건강 & 영양소 체크

> 1~2분 자가 건강체크를 통해 부족한 영양소와 의심 증상을 확인하는 웹 애플리케이션

## 🎯 프로젝트 개요

약국이나 병원에서 대기 중인 환자들이 **1~2분 내외**로 간단하게 자가 건강체크를 수행하고, 
자신에게 부족한 영양소와 관련 증상을 직관적으로 확인할 수 있는 모바일 최적화 웹앱입니다.

## ✅ 구현된 기능

### 1. 기본 정보 입력 (STEP 1)
- **성별** 선택 (남성/여성)
- **나이대** 선택 (10대 ~ 60대 이상, 6단계)
- **생활 패턴** 선택 (앉아서 생활 ~ 매우 활동적, 4단계)
- **식습관 유형** 선택 (일반식, 육류위주, 탄수화물위주, 불규칙, 채식 5가지)

### 2. 증상 체크리스트 (STEP 2~8) - 7개 카테고리, 22개 문항
| 카테고리 | 문항 수 | 주요 체크 내용 |
|---------|--------|--------------|
| 에너지 & 수면 | 4문항 | 피로감, 수면 질, 식후 졸음, 아침 기상 |
| 피부 & 모발 | 3문항 | 피부 상태(복수), 모발(복수), 손톱 |
| 소화 & 장 건강 | 2문항 | 소화 상태(복수), 식욕 |
| 정신건강 & 인지 | 3문항 | 기분, 집중력, 스트레스 |
| 근골격 & 관절 | 3문항 | 근육 경련, 관절통, 근력 |
| 면역 & 눈 & 구강 | 3문항 | 면역력, 눈(복수), 구강(복수) |
| 생활습관 & 기타 | 5문항 | 햇빛 노출, 음주, 카페인, 수분, 흡연 |

- 모든 문항 **객관식** (단일선택 또는 복수선택)
- 이모지 아이콘으로 직관적 선택지
- 카테고리 탭으로 자유로운 이동 가능

### 3. 영양소 결핍 분석 엔진
- **규칙 기반 전문가 시스템** (Rule-Based Expert System)
- 30개 이상 영양소 매핑 데이터
- 4단계 분석 파이프라인:
  1. 증상 기반 기본 점수 계산
  2. 인구통계학적 보정 (나이/성별/생활습관/식습관)
  3. 복합 결핍 시너지 패턴 인식 (빈혈 패턴, 뼈 건강 패턴 등)
  4. 최종 결과 생성 및 등급 산출

### 4. 결과 페이지
- **영양 균형 점수** (100점 만점, A~F 등급)
- **레이더 차트** (카테고리별 건강 상태 시각화)
- **부족 의심 영양소 TOP 10** (심각도별 색상 구분)
  - 각 영양소별: 설명, 관련 증상, 권장 식품, 일일 권장량, 상세 정보
- **맞춤 건강 조언** (프로필 기반 개인화된 조언)
- 결과 공유 기능 (Web Share API / 클립보드 복사)
- 다시 체크하기 기능

### 5. UX/UI 특징
- 모바일 퍼스트 반응형 디자인
- 상단 진행 프로그레스 바
- 하단 고정 네비게이션 (이전/다음)
- 부드러운 애니메이션 전환
- 미선택 시 토스트 알림

## 📁 파일 구조

```
index.html              - 메인 HTML (스텝별 섹션 구조)
css/
  └── style.css         - 전체 스타일시트 (애니메이션, 컴포넌트)
js/
  ├── checklist-data.js - 체크리스트 데이터 (7카테고리, 22문항, 30+영양소 정보)
  ├── analysis-engine.js - 영양소 결핍 분석 엔진 (NutrientAnalysisEngine 클래스)
  └── app.js            - 메인 앱 로직 (네비게이션, UI렌더링, 결과 표시)
```

## 🛠️ 기술 스택

- **HTML5** + **Tailwind CSS** (CDN)
- **Vanilla JavaScript** (프레임워크 없음)
- **Chart.js** (레이더 차트)
- **Font Awesome 6** (아이콘)
- **Noto Sans KR** (한국어 폰트)

## 🔮 향후 개선 사항

1. **결과 히스토리**: 이전 체크 결과 저장 및 비교 기능
2. **PDF 출력**: 결과를 PDF로 다운로드하는 기능
3. **약사/의사 연동**: 결과를 의료진에게 바로 공유하는 QR코드 기능
4. **다국어 지원**: 영어, 중국어 등 다국어 체크리스트
5. **심화 체크**: 2차 상세 진단 (혈액검사 결과 입력 연동)
6. **보충제 추천**: 결핍 영양소 기반 보충제 제품 추천
7. **계절/시기별 맞춤**: 계절, 생리주기 등 시기적 변수 반영
